Das Verzeichnis Examples enthält Beispiele zur Programmierung der
isel-CAN-CNC-Steuerung.


Visual C++ 6.0 Beispiele

  Stellen hierzu Sie in Visual C++ Studio 

  ein include-Verzeichnis auf %Installationsordner%\control\can\include
  ein lib-Verzeichnis auf %Installationsordner%\control\can\lib

  (Siehe Extras->Optionen) im Visual C++ Studio

  Gegegebenfalls muss für Ihre Anwendung der Pfad auf 
  %Installationsordner%\control\can gestellt werden.



  Die Beispiele finden Sie in %Installationsordner%\control\can\examples

  Im Allgemeinen ist %Installationsordner% = C:\cncworkbench

Beispiele:

  proncdllsdo           : Beispiel zum Senden eines SDOs in ProNC über eine DLL.
  mctl1                 : Beispiel für eine Bewegung in X- und Y-Richtung
  axctlexampleseriell   : Beispiel zur Ansteuerung einer Achse über die
                          serielle Schnittstelle
  VisualBasic           : Beispiele in Visual Basic

Visual Studio 2008:

  AxctlCSharpExample.zip: Beispiel Handlingachse in CSharp
